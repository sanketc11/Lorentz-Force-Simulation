# Lorentz Force Simulation ⚡

A Python real-time simulation of charged particle motion in a magnetic field using the Lorentz force.

## Demo

A charged particle moving through a magnetic field:

![Lorentz Force Simulation](https://github.com/sanketc11/Lorentz-Force-Simulation/blob/main/assets/simulation.gif)

## Overview

This simulation shows how a charged particle moves when affected by a magnetic field.

The magnetic Lorentz force is:

`F = q(v × B)`

The force acts perpendicular to the particle's velocity, causing curved trajectories.

## Features

- Real-time particle motion
- Magnetic field visualization
- Particle trajectory tracking
- Vector-based physics calculations
